# 📊 Amazon Sales Dashboard | Power BI Project

## 🚀 Project Overview
This project is a **Power BI Dashboard** designed to analyze Amazon sales data and order details.  
It helps to understand sales performance, order trends, customer behavior, and product insights through interactive visualizations 📈

---

## 🧾 Order Details Insights
This dashboard focuses on analyzing key order-level metrics such as:

- 📦 Total Orders placed  
- 💰 Total Sales Revenue  
- 🛒 Product-wise order distribution  
- 🚚 Order status (Delivered, Pending, Returned)  
- 📅 Daily, Monthly, Yearly sales trends  
- 🌍 Region-wise performance analysis  
- 👤 Customer purchasing behavior  

---

## 📊 Key Features
- 📌 Interactive Power BI dashboard  
- 📊 KPI cards for quick insights  
- 📈 Trend analysis using charts and graphs  
- 🗂️ Filters for Category, Region, and Date  
- 🔍 Drill-down capability for deeper analysis  
- ⚡ Clean and professional dashboard design  

---

## 🛠️ Tools & Technologies Used
- 📊 Power BI  
- 🧹 Power Query (Data Cleaning & Transformation)  
- 🧮 DAX (Data Analysis Expressions)  
- 📁 Excel / CSV Dataset  

---

## 📌 Business Insights
This dashboard helps in understanding:

- 🏆 Best-selling products  
- 🌍 High-performing regions  
- 🚚 Delivery and order efficiency  
- 🧠 Customer buying patterns  
- 📈 Revenue growth trends
